Commonly used Git commands.
Remote repository commands:
    git clone git@github.com:USER-NAME/REPOSITORY-NAME.git
    git push or git push origin main
Workflow commands:
    git add .
    git commit -m "A message describing what you have done to make snapshot different"
Check Status or Log history:
    git Status
    git Log
Git Syntax:
    program | action | destination
        "PAD" ACRONYM

Quick Look at Process;

Create Repo on Github Account
    -Select Code and copy SSH URL link
    -command git clone followed by URL link in repos folder on CLI
        (connects repo on GitHub to local machine)
    -command cd into new project folder
    -command git remote -v to display URL of Repo

Commands to modify, stage and commit
    -add file or directory, save as neccessary
    -get status (red and being modified)
    -command git add file (green and being staged)
    -command git commit file or directory (.)
    -command git status
    -command git log
